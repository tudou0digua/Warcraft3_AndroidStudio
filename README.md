# Warcraft3_AndroidStudio
魔兽争霸3资料查询手册   
1.该手册使用AndroidStudio开发，实时证明AndroidStudio正式版的开发效率比用Eclipse要高。  
2.该手册包含魔兽争霸3中人族、暗夜、中立、兽族和不死族所有的英雄、单位和建筑资料。
